# WORLDr — Voter Blocs Reference (Keldoria)

## Overview

Voter blocs are political demographic segments. Unlike population groups (which drive economic simulation), voter blocs drive political mechanics: party support, election outcomes, turnout, and approval dynamics.

Keldoria has 12 voter blocs representing its 82 million citizens.

## Approval Calculation Logic

Each voter bloc's approval is calculated as:

```
bloc_approval = base_approval
  - (inflation_rate × inflation_sensitivity × inflation_weight)
  - (unemployment_rate × unemployment_sensitivity × unemployment_weight)
  + (welfare_spending_index × welfare_weight × bloc_welfare_dependence)
  + (party_affinity_bonus if governing party matches bloc ideology)
  ± (active_law_effects for this bloc)
```

The **weighted national approval** = Σ(bloc_approval × bloc_population_share × bloc_turnout_weight)
