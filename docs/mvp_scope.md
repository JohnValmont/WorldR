# WORLDr MVP Scope

Goal:
Create a playable single-nation domestic political/economic simulator.

MVP includes:
- authentication
- nation state
- GDP simulation
- sectors
- inflation
- taxes
- budget
- laws
- approval
- historical charts
- monthly ticks

MVP excludes:
- diplomacy
- military
- multiplayer ministries
- world map
- trade system
- espionage
- war
- AI nations
- elections
- media
- corruption

Success condition:
Player can manage one nation for multiple years and experience realistic economic and political consequences.
